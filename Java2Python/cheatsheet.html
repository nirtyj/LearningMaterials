📝 Quick Cheat Sheet

Side-by-side syntax reference

Operation Python Java Equivalent
Print print(x) System.out.println(x)
String length len(s) s.length()
Array/List length len(arr) arr.length / list.size()
Char at index s[i] s.charAt(i)
Substring s[i:j] s.substring(i, j)
String → Int int(s) Integer.parseInt(s)
Int → String str(n) String.valueOf(n)
Max/Min max(a, b) Math.max(a, b)
Absolute value abs(x) Math.abs(x)
Power x ** y Math.pow(x, y)
Floor division a // b Math.floorDiv(a, b)
Check type isinstance(x, int) x instanceof Integer
Null/None check x is None x == null
Boolean AND/OR and / or && / ||
Not equal != != (primitives) / !equals()
