📝 String

Immutable in both languages — and the building patterns that follow

Operations

🐍 Python
s = "hello world"

# ── Access ──
s[0]                         # 'h'
s[-1]                        # 'd'
s[1:5]                       # 'ello' (slicing)

# ── Modify (strings are immutable!) ──
s.upper()                    # 'HELLO WORLD'
s.lower()                    # 'hello world'
s.strip()                    # trim whitespace
s.replace("hello", "hi")    # 'hi world'

# ── Split / Join ──
s.split()                    # ['hello','world']
s.split(",")                 # split on comma
" ".join(["a", "b"])        # "a b"

# ── Check ──
s.startswith("he")          # True
s.endswith("ld")             # True
s.isdigit()                  # False
s.isalpha()                  # False (has space)
"ell" in s                  # True (substring check)
s.find("lo")                # 3 (-1 if not found)

# ── Reverse ──
s[::-1]                      # 'dlrow olleh'

# ── Build string efficiently ──
chars = []
chars.append("h")
chars.append("i")
result = "".join(chars)      # "hi"

# ── f-strings ──
name = "Bob"
f"Hello {name}, 2+2={2+2}"  # "Hello Bob, 2+2=4"
☕ Java
String s = "hello world";

// ── Access ──
s.charAt(0);                 // 'h'
s.charAt(s.length()-1);      // 'd'
s.substring(1, 5);           // "ello"

// ── Modify (strings are immutable!) ──
s.toUpperCase();             // "HELLO WORLD"
s.toLowerCase();             // "hello world"
s.trim();                    // trim whitespace
s.replace("hello", "hi");   // "hi world"

// ── Split / Join ──
s.split("\\s+");              // ["hello","world"]
s.split(",");                // split on comma
String.join(" ", "a", "b"); // "a b"

// ── Check ──
s.startsWith("he");          // true
s.endsWith("ld");             // true
Character.isDigit(c);        // per char
Character.isLetter(c);       // per char
s.contains("ell");           // true
s.indexOf("lo");             // 3 (-1 if not found)

// ── Reverse ──
new StringBuilder(s).reverse().toString();

// ── Build string efficiently ──
StringBuilder sb = new StringBuilder();
sb.append("h");
sb.append("i");
String result = sb.toString(); // "hi"

// ── String formatting ──
String name = "Bob";
String.format("Hello %s, 2+2=%d", name, 4);

🛠️ Built-ins commonly paired with strings

When you write string code in interviews, these are the built-ins you reach for most.

# ── Char access ──
s[0], s[-1], s[i:j]              # indexing, slicing — no charAt() ceremony
ord(c), chr(97)                  # char ↔ int  (Java's c - 'a' bucket: ord(c) - ord('a'))

# ── Char checks (per-char or whole-string) ──
c.isalnum(), c.isalpha()         # letter or digit / letter
c.isdigit(), c.isspace()         # digit / whitespace
c.islower(), c.isupper()         # case
c.lower(), c.upper(), c.casefold()  # case-fold for compares

# ── Iterating ──
for c in s: ...                  # direct char iteration
for i, c in enumerate(s): ...    # index + char
for a, b in zip(s, s[1:]): ...   # adjacent pairs (palindrome scans, runs)
for a, b in pairwise(s): ...     # 3.10+ — same idea via itertools

# ── Transforming ──
"".join(reversed(s))             # reverse without slicing
"".join(c for c in s if c.isalnum())  # filter chars in one shot
[c.lower() for c in s if c.isalnum()] # cleaned list of chars

# ── Searching / aggregating ──
s.count("ab")                    # non-overlapping count
sum(1 for c in s if c.isdigit()) # count by predicate
max(s.split(), key=len)          # longest word

Immutability — Why Building With list Matters

String Immutability

# Strings are immutable in BOTH languages
s = "hello"
# s[0] = "H"  # ERROR!

# ✅ Create new string
s = "H" + s[1:]

# For many modifications, use list
chars = list(s)
chars[0] = "H"
s = "".join(chars)

Repeated s += "x" in a loop is O(n²) in Python — every += allocates a fresh string. Build a list and "".join() at the end for O(n).
