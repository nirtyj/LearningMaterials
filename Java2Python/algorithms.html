🧩 Algorithm Patterns

Pattern-recognition cheat sheet — Python tricks for the Neetcode 150 / Blind 75

Patterns tied to a specific data structure (Monotonic Stack, BFS, Heap/Top-K) live on those pages instead — see Stack, Queue, Heap.

🗺️ Pattern Selection Flowchart

  • Input array is sorted? → Binary Search or Two Pointers
  • Asked for all permutations / combinations / subsets? → Backtracking
  • Given a tree or graph? → DFS / BFS
  • Given a linked list? → Two Pointers (fast/slow)
  • Asked for top/least K items? → Heap
  • Asked for max/min subarray? → DP or Kadane's / Sliding Window
  • Dealing with contiguous subarray/substring? → Sliding Window
  • Need O(1) lookup? → Hash Map / Set
  • Finding connected components? → Union Find or DFS
  • Scheduling / overlapping intervals? → Sort + Greedy
  • Need prefix string search? → Trie
  • Next greater/smaller element? → Monotonic Stack

1️⃣ Prefix Sum

Precompute cumulative sums for O(1) range-sum queries. Used in: Subarray Sum Equals K, Range Sum Query, Product of Array Except Self

🐍 Python Shortcuts
# ── Build prefix sum ──
from itertools import accumulate
prefix = list(accumulate(nums, initial=0))
# range sum [i, j] = prefix[j+1] - prefix[i]

# ── Manual prefix sum ──
prefix = [0] * (len(nums) + 1)
for i, n in enumerate(nums):
    prefix[i + 1] = prefix[i] + n

# ── Subarray Sum Equals K (LC 560) ──
# Trick: use a dict to count prefix sums seen so far
from collections import defaultdict
def subarraySum(nums, k):
    count = cur = 0
    seen = defaultdict(int)
    seen[0] = 1                          # ← empty prefix
    for n in nums:
        cur += n
        count += seen[cur - k]           # ← O(1) lookup
        seen[cur] += 1
    return count

# ── Product Except Self (LC 238) ──
# Trick: build left and right products in-place
def productExceptSelf(nums):
    n = len(nums)
    ans = [1] * n
    for i in range(1, n):
        ans[i] = ans[i-1] * nums[i-1]     # left products
    right = 1
    for i in range(n-1, -1, -1):
        ans[i] *= right
        right *= nums[i]                 # right products
    return ans

2️⃣ Two Pointers

Opposite ends converging or same-direction pointers on sorted arrays. Used in: Two Sum II, 3Sum, Container With Most Water, Trapping Rain Water, Valid Palindrome

🐍 Python Shortcuts
# ── Template: Opposite-end pointers ──
l, r = 0, len(nums) - 1
while l < r:
    if condition:
        l += 1
    else:
        r -= 1

# ── 3Sum (LC 15) ──
# Trick: sort + skip duplicates with while loops
def threeSum(nums):
    nums.sort()
    res = []
    for i in range(len(nums) - 2):
        if i > 0 and nums[i] == nums[i-1]:
            continue                     # ← skip duplicates
        l, r = i + 1, len(nums) - 1
        while l < r:
            s = nums[i] + nums[l] + nums[r]
            if s < 0: l += 1
            elif s > 0: r -= 1
            else:
                res.append([nums[i], nums[l], nums[r]])
                while l < r and nums[l] == nums[l+1]: l += 1
                l += 1; r -= 1
    return res

# ── Valid Palindrome (LC 125) ──
# Trick: isalnum() + two pointers
def isPalindrome(s):
    s = [c.lower() for c in s if c.isalnum()]
    return s == s[::-1]                  # ← one-liner!

# ── Python tricks used ──
# • a, b = b, a       — swap without temp
# • s.isalnum()        — checks alphanumeric
# • nums.sort()        — in-place O(n log n)
# • s == s[::-1]       — palindrome check

3️⃣ Sliding Window

Expand/shrink a window over contiguous elements. Used in: Best Time to Buy/Sell Stock, Longest Substring Without Repeating Chars, Minimum Window Substring, Permutation in String

🐍 Python Shortcuts
# ── Template: Variable-size window ──
def sliding_window(s):
    window = defaultdict(int)            # ← track window state
    l = res = 0
    for r in range(len(s)):
        window[s[r]] += 1               # expand
        while window_invalid():           # shrink
            window[s[l]] -= 1
            l += 1
        res = max(res, r - l + 1)       # update answer
    return res

# ── Longest Substring Without Repeating (LC 3) ──
def lengthOfLongestSubstring(s):
    seen = {}                            # char → last index
    l = res = 0
    for r, c in enumerate(s):          # ← enumerate!
        if c in seen and seen[c] >= l:
            l = seen[c] + 1
        seen[c] = r
        res = max(res, r - l + 1)
    return res

# ── Minimum Window Substring (LC 76) ──
# Trick: Counter for need vs window tracking
def minWindow(s, t):
    need = Counter(t)
    missing = len(t)
    l = start = 0; end = float('inf')
    for r, c in enumerate(s):
        if need[c] > 0: missing -= 1
        need[c] -= 1
        while missing == 0:              # all chars found
            if r - l < end - start:
                start, end = l, r
            need[s[l]] += 1
            if need[s[l]] > 0: missing += 1
            l += 1
    return s[start:end+1] if end < float('inf') else ""

# ── Python tricks used ──
# • Counter(t)         — instant frequency map
# • defaultdict(int)   — no KeyError on missing keys
# • enumerate(s)       — index + char in one shot
# • float('inf')       — easy infinity comparison

4️⃣ Fast & Slow Pointers

Detect cycles, find middle of linked list. Used in: Linked List Cycle, Middle of Linked List, Happy Number, Find Duplicate Number

🐍 Python Shortcuts
# ── Detect cycle (LC 141) ──
def hasCycle(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow == fast: return True
    return False

# ── Find middle (LC 876) ──
def middleNode(head):
    slow = fast = head
    while fast and fast.next:
        slow, fast = slow.next, fast.next.next  # ← parallel assign!
    return slow

# ── Happy Number (LC 202) ──
# Trick: treat as cycle detection in number sequence
def isHappy(n):
    def next_num(n):
        return sum(int(d)**2 for d in str(n))  # ← str→digits trick!
    slow, fast = n, next_num(n)
    while fast != 1 and slow != fast:
        slow = next_num(slow)
        fast = next_num(next_num(fast))
    return fast == 1

5️⃣ Binary Search

Halve search space each step. Used in: Binary Search, Search in Rotated Array, Find Min in Rotated Array, Koko Eating Bananas, Search 2D Matrix

🐍 Python Shortcuts
# ── Template: standard binary search ──
def binary_search(nums, target):
    l, r = 0, len(nums) - 1
    while l <= r:
        m = (l + r) // 2               # ← no overflow in Python!
        if nums[m] == target: return m
        elif nums[m] < target: l = m + 1
        else: r = m - 1
    return -1

# ── bisect module (built-in binary search!) ──
from bisect import bisect_left, bisect_right, insort
i = bisect_left(arr, target)         # leftmost insert point
i = bisect_right(arr, target)        # rightmost insert point
insort(arr, val)                     # insert keeping sorted
# Check if target exists:
found = i < len(arr) and arr[i] == target

# ── Binary Search on Answer Space (LC 875 Koko) ──
# Trick: binary search on the answer, not the array
import math
def minEatingSpeed(piles, h):
    l, r = 1, max(piles)
    while l < r:
        m = (l + r) // 2
        if sum(math.ceil(p / m) for p in piles) <= h:
            r = m                        # ← can eat slower
        else:
            l = m + 1
    return l

# ── Python tricks used ──
# • No integer overflow (Python has big ints!)
# • bisect_left/right  — skip writing binary search
# • math.ceil(a/b) or -(-a//b)  — ceiling division
# • sum(... for ... in ...)  — generator expression

🌳 SortedList — Python's missing TreeMap

Python's stdlib has no balanced BST / TreeMap / TreeSet. sortedcontainers.SortedList fills that hole — O(log n) insert + delete + indexed access in pure Python. It's pre-installed on LeetCode and most online judges, and is the standard answer for sliding-window-median, count-of-smaller, and order-statistic problems. 

from sortedcontainers import SortedList, SortedDict, SortedSet

sl = SortedList([5, 1, 3])
sl.add(2)                          # O(log n)  → [1, 2, 3, 5]
sl.remove(3)                       # O(log n)  → [1, 2, 5]
sl[0], sl[-1]                     # O(log n) indexed access — min & max
sl.bisect_left(3)                  # index of first ≥ 3
sl.bisect_right(3)                 # index of first > 3
sl.irange(2, 5)                    # iterate values in [2, 5] inclusive

# ── Sliding Window Median (LC 480) ──
# Plain list: O(k) insert per slide → TLE.
# SortedList: O(log k) insert + O(1) median by index.
def medianSlidingWindow(nums, k):
    window = SortedList(nums[:k])
    res = []
    for i in range(k, len(nums) + 1):
        res.append((window[k//2] + window[~(k//2)]) / 2)  # ← ~i trick!
        if i < len(nums):
            window.remove(nums[i - k])
            window.add(nums[i])
    return res

# ── Patience-sort LIS (LC 300) — bisect on a SortedList alt ──
# Plain bisect+list works here because we only overwrite, never insert middle:
from bisect import bisect_left
def lengthOfLIS(nums):
    tails = []                          # tails[i] = smallest tail of any inc subseq of length i+1
    for n in nums:
        i = bisect_left(tails, n)
        if i == len(tails):
            tails.append(n)             # extend longest run
        else:
            tails[i] = n                # replace — keeps tails sorted
    return len(tails)                 # O(n log n)

# ── Count of Smaller Numbers After Self (LC 315) ──
# Walk right→left, ask "how many already-seen are < current?"
def countSmaller(nums):
    seen = SortedList()
    res = []
    for n in reversed(nums):
        res.append(seen.bisect_left(n))  # O(log n)
        seen.add(n)
    return res[::-1]

🎯 When to reach for it

  • Need insert + delete + min/max all in O(log n) → SortedList (heap can't delete arbitrary)
  • Need order statistics (the k-th smallest) → SortedList (heap can't index)
  • Need range queries on a sorted multiset → SortedList.irange()
  • Plain bisect + list is fine when you only append/replace at known indices (LIS) — the O(n) insert middle is what kills you, and that's exactly what SortedList avoids
  • Interview note: most interviewers accept SortedList at FAANG. If forbidden, the fallback is two heaps (median) or a manually-balanced BST / segment tree

def reverseList(head):
    prev, curr = None, head
    while curr:
        curr.next, prev, curr = prev, curr, curr.next  # ← triple swap!
    return prev

# ── Merge two sorted lists (LC 21) ──
def mergeTwoLists(l1, l2):
    dummy = cur = ListNode(0)
    while l1 and l2:
        if l1.val < l2.val:
            cur.next, l1 = l1, l1.next
        else:
            cur.next, l2 = l2, l2.next

7️⃣ DFS / Tree Traversal

Recursive depth-first exploration. Used in: Max Depth, Same Tree, Invert Tree, Diameter, Lowest Common Ancestor, Serialize/Deserialize, Number of Islands

🐍 Python Shortcuts
# ── Max Depth (LC 104) ──
def maxDepth(root):
    if not root: return 0
    return 1 + max(maxDepth(root.left), maxDepth(root.right))

# ── Diameter of Binary Tree (LC 543) ──
# Trick: nonlocal to track state across recursive calls
def diameterOfBinaryTree(root):
    res = 0
    def dfs(node):
        nonlocal res                    # ← access outer variable
        if not node: return 0
        l, r = dfs(node.left), dfs(node.right)
        res = max(res, l + r)
        return 1 + max(l, r)
    dfs(root)
    return res

# ── Number of Islands (LC 200) — grid DFS ──
def numIslands(grid):
    def dfs(r, c):
        if r < 0 or r >= rows or c < 0 or c >= cols \
           or grid[r][c] != "1":
            return
        grid[r][c] = "#"                # ← mark visited in-place
        for dr, dc in [(1,0),(-1,0),(0,1),(0,-1)]:
            dfs(r + dr, c + dc)
    rows, cols = len(grid), len(grid[0])
    count = 0
    for r in range(rows):
        for c in range(cols):
            if grid[r][c] == "1":
                dfs(r, c)
                count += 1
    return count

# ── Python tricks used ──
# • nonlocal            — modify enclosing scope variable
# • nested def          — closures capture outer vars
# • modify grid in-place — no need for visited set
# • for dr, dc in [...]  — clean direction iteration

8️⃣ Backtracking

Explore all combinations/permutations, undo choices. Used in: Subsets, Permutations, Combination Sum, Word Search, N-Queens, Generate Parentheses

🐍 Python Shortcuts
# ── Subsets (LC 78) ──
def subsets(nums):
    res = []
    def backtrack(start, path):
        res.append(path[:])              # ← copy! (path[:] or list(path))
        for i in range(start, len(nums)):
            path.append(nums[i])
            backtrack(i + 1, path)
            path.pop()                   # ← undo choice (backtrack)
    backtrack(0, [])
    return res

# ── Permutations (LC 46) ──
def permute(nums):
    res = []
    def backtrack(path):
        if len(path) == len(nums):
            res.append(path[:])
            return
        for n in nums:
            if n not in path:            # ← O(n) but clean for interviews
                path.append(n)
                backtrack(path)
                path.pop()
    backtrack([])
    return res

# ── Combination Sum (LC 39) ──
def combinationSum(candidates, target):
    res = []
    def backtrack(start, path, total):
        if total == target:
            res.append(path[:])
            return
        for i in range(start, len(candidates)):
            if total + candidates[i] > target:
                break                    # ← prune (sort first!)
            path.append(candidates[i])
            backtrack(i, path, total + candidates[i])  # i not i+1 (reuse)
            path.pop()
    candidates.sort()                    # ← sort enables pruning
    backtrack(0, [], 0)
    return res

# ── Python tricks used ──
# • path[:]            — shallow copy of current path
# • path.append / pop  — mutate-then-undo pattern
# • sort + break       — prune search space
# • n not in path      — membership test in list

💡 Library shortcut: itertools.combinations(arr, r), itertools.permutations(arr), and itertools.product(arr, repeat=n) generate these exhaustively without the manual template — fine for vanilla "produce all subsets/permutations" prompts. Keep the manual template ready for variants with constraints (early pruning, custom state, "stop at first match"). Full menu: Python Builtins → itertools.

9️⃣ Intervals

Sort + merge/compare overlapping ranges. Used in: Merge Intervals, Insert Interval, Non-Overlapping Intervals, Meeting Rooms

# ── Merge Intervals (LC 56) ──
def merge(intervals):
    intervals.sort()                     # ← sorts by start (then end)
    merged = [intervals[0]]
    for start, end in intervals[1:]:    # ← tuple unpacking
        if start <= merged[-1][1]:
            merged[-1][1] = max(merged[-1][1], end)
        else:
            merged.append([start, end])
    return merged

# ── Meeting Rooms II (sweep line) ──
# Trick: separate start/end events and sort
events = []
for s, e in intervals:
    events.append((s, 1))               # meeting starts
    events.append((e, -1))              # meeting ends
events.sort()
max_rooms = cur = 0
for _, delta in events:
    cur += delta
    max_rooms = max(max_rooms, cur)

🔟 Dynamic Programming

Optimal substructure + overlapping subproblems. Used in: Climbing Stairs, House Robber, Coin Change, LIS, Longest Common Subsequence, Knapsack, Edit Distance, Word Break

🐍 Python Shortcuts
# ── Top-Down with @cache (Python 3.9+) ──
from functools import cache
def climbStairs(n):
    @cache                               # ← automatic memoization!
    def dp(i):
        if i <= 1: return 1
        return dp(i-1) + dp(i-2)
    return dp(n)

# ── For Python < 3.9, use @lru_cache(maxsize=None) ──
from functools import lru_cache
@lru_cache(maxsize=None)
def dp(i, j): ...

# ── Bottom-Up: Coin Change (LC 322) ──
def coinChange(coins, amount):
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0
    for a in range(1, amount + 1):
        for c in coins:
            if c <= a:
                dp[a] = min(dp[a], dp[a - c] + 1)
    return dp[amount] if dp[amount] != float('inf') else -1

# ── House Robber (LC 198) — space-optimized ──
def rob(nums):
    prev, curr = 0, 0
    for n in nums:
        prev, curr = curr, max(curr, prev + n)  # ← parallel assign!
    return curr

# ── LIS (LC 300) — patience sort with bisect ──
from bisect import bisect_left
def lengthOfLIS(nums):
    tails = []
    for n in nums:
        i = bisect_left(tails, n)
        if i == len(tails):
            tails.append(n)
        else:
            tails[i] = n                 # ← O(n log n)!
    return len(tails)

# ── 2D DP: LCS (LC 1143) ──
def longestCommonSubsequence(s1, s2):
    dp = [[0] * (len(s2)+1) for _ in range(len(s1)+1)]
    for i in range(1, len(s1)+1):
        for j in range(1, len(s2)+1):
            if s1[i-1] == s2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    return dp[-1][-1]                    # ← dp[-1][-1] = bottom-right

# ── Python tricks used ──
# • @cache / @lru_cache  — FREE memoization decorator
# • float('inf')         — clean infinity for min comparisons
# • prev, curr = curr, max(...)  — O(1) space DP
# • bisect_left          — O(n log n) LIS
# • dp[-1][-1]           — negative indexing for last element

1️⃣1️⃣ Graph / Topological Sort

Adjacency list traversal, dependency ordering. Used in: Course Schedule, Clone Graph, Pacific Atlantic Water Flow, Alien Dictionary

🐍 Python Shortcuts
# ── Build adjacency list ──

# ── Topological Sort — BFS / Kahn's (LC 207) ──
def canFinish(numCourses, prerequisites):
    graph = defaultdict(list)
    indegree = [0] * numCourses
    for course, pre in prerequisites:
        graph[pre].append(course)
        indegree[course] += 1
    q = deque(i for i in range(numCourses) if indegree[i] == 0)
    taken = 0                            # ← generator in deque!
    while q:
        node = q.popleft()
        taken += 1
        for nei in graph[node]:
            indegree[nei] -= 1
            if indegree[nei] == 0:
                q.append(nei)
    return taken == numCourses

# ── Topological Sort — DFS cycle detection ──
def canFinish_dfs(numCourses, prerequisites):
    graph = defaultdict(list)
    for c, p in prerequisites: graph[p].append(c)
    WHITE, GRAY, BLACK = 0, 1, 2       # ← state enum
    color = [WHITE] * numCourses
    def dfs(node):
        color[node] = GRAY
        for nei in graph[node]:
            if color[nei] == GRAY: return False  # cycle!
            if color[nei] == WHITE and not dfs(nei): return False
        color[node] = BLACK
        return True
    return all(dfs(i) for i in range(numCourses) if color[i] == WHITE)

# ── Python tricks used ──
# • defaultdict(list)   — no KeyError, auto-creates adj list
# • deque(generator)    — initialize queue with generator
# • all(... for ...)    — short-circuit over all nodes
# • WHITE, GRAY, BLACK  — multiple assignment for states

1️⃣2️⃣ Trie (Prefix Tree)

Efficient prefix/word lookup. Used in: Implement Trie, Word Search II, Design Add and Search Words

🐍 Python Shortcuts
# ── Clean Trie using nested dicts ──
class Trie:
    def __init__(self):
        self.root = {}

    def insert(self, word):
        node = self.root
        for c in word:
            node = node.setdefault(c, {}) # ← create if missing!
        node['#'] = True                  # ← end-of-word marker

    def search(self, word):
        node = self.root
        for c in word:
            if c not in node: return False
            node = node[c]
        return '#' in node

    def startsWith(self, prefix):
        node = self.root
        for c in prefix:
            if c not in node: return False
            node = node[c]
        return True

# ── Python tricks used ──
# • dict.setdefault(k, {})  — get-or-create in one call
# • nested dict as trie     — no need for TrieNode class
# • '#' as end marker       — any sentinel works

1️⃣3️⃣ Union Find (Disjoint Set)

Track connected components efficiently. Used in: Number of Connected Components, Redundant Connection, Accounts Merge, Number of Provinces

🐍 Python Shortcuts
# ── Template: Union Find with path compression + rank ──
class UnionFind:
    def __init__(self, n):
        self.par = list(range(n))       # ← parent[i] = i initially
        self.rank = [0] * n

    def find(self, x):
        if self.par[x] != x:
            self.par[x] = self.find(self.par[x])  # ← path compression
        return self.par[x]

    def union(self, x, y):
        px, py = self.find(x), self.find(y)
        if px == py: return False         # already connected
        if self.rank[px] < self.rank[py]:
            px, py = py, px              # ← swap trick
        self.par[py] = px
        if self.rank[px] == self.rank[py]:
            self.rank[px] += 1
        return True

# ── Usage: Number of Connected Components ──
uf = UnionFind(n)
for u, v in edges:
    uf.union(u, v)
components = len(set(uf.find(i) for i in range(n)))  # ← set comp!

# ── Python tricks used ──
# • list(range(n))       — initialize parent array
# • recursive find + reassign — path compression
# • set(... for ...)     — count unique roots

⚙️ Bit Tricks for Tight Loops — applied patterns

The one-liners above are the vocabulary. These are the sentences — three patterns that turn bit tricks into algorithms: Kernighan popcount, Fenwick lowbit, and bitmask-DP state. 

# ── 1) Kernighan's algorithm — popcount in O(set bits) ──
# Faster than bin(n).count('1') when bits are sparse, and the
# interview-favorite "count 1 bits without bin()" answer.
def popcount(n):
    count = 0
    while n:
        n &= n - 1                       # ← drops the lowest set bit
        count += 1
    return count

# ── 2) Fenwick / Binary Indexed Tree — i & -i is the engine ──
# Used in: Range Sum Query Mutable, Count of Smaller Numbers After Self.
class BIT:
    def __init__(self, n):
        self.tree = [0] * (n + 1)
    def update(self, i, delta):
        while i < len(self.tree):
            self.tree[i] += delta
            i += i & -i                  # ← jump to next responsible node
    def query(self, i):                  # prefix sum [1..i]
        s = 0
        while i > 0:
            s += self.tree[i]
            i -= i & -i                  # ← peel off lowest set bit
        return s

# Used in: Travelling Salesman, Partition to K Equal Subsets,
# Shortest Path Visiting All Nodes, Stickers to Spell Word.
full = (1 << n) - 1                     # mask with all n bits set
for mask in range(1 << n):                # every subset of {0..n-1}
    for i in range(n):

🎯 When each pays off

  • Kernighan beats bin(n).count('1') only inside hot loops on sparse n — for one-shot popcount, just use bin(...)
  • Fenwick lowbit pattern memorize: i += i & -i climbs the tree (update), i -= i & -i walks toward root (query)
  • Bitmask DP works when n ≤ 20 — 2^20 = ~1M states still fits memory; 2^22+ usually doesn't
  • format(n, '0{w}b') is cleaner than bin(n)[2:].zfill(w) for fixed-width — same result, no string math

# ── Maximum Subarray / Kadane's (LC 53) ──
def maxSubArray(nums):
    cur = res = nums[0]
    for n in nums[1:]:
        cur = max(n, cur + n)            # ← restart or extend?
        res = max(res, cur)
    return res

# ── Jump Game (LC 55) ──
def canJump(nums):
    reach = 0
    for i, n in enumerate(nums):
        if i > reach: return False
        reach = max(reach, i + n)
    return True

# ── Partition Labels (LC 763) ──
def partitionLabels(s):
    last = {c: i for i, c in enumerate(s)}  # ← dict comp!
    start = end = 0
    res = []
    for i, c in enumerate(s):
        end = max(end, last[c])
        if i == end:
            res.append(end - start + 1)
            start = i + 1
    return res
