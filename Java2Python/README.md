This folder has resources to learn about moving from Java to Python for Interviews only.

## Visual Guide (multi-page)

Open **[`index.html`](./index.html)** in a browser for the visual guide. Each topic is its own page so you can read about a single data structure end-to-end (comparison + initialization + operations + interview pattern):

- **Foundations** — Compilation & Execution, Variables & Memory, Time Complexity
- **Data Structures** — List, Dict, Set, Stack, Queue, Heap, String, DefaultDict / Counter
- **OOP & Classes** — node defs, dataclass, dunder, sorting, abstract, OOD building blocks
- **Patterns & Reference** — Algorithm Patterns, Python Power Features, Common Gotchas, Quick Cheat Sheet

The legacy single-page version is still available at `java_to_python_guide.html` for reference.
