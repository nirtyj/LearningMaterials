The Convolution Operation

A small filter slides across the image computing a dot product at every position.

Like a flashlight scanning the image, a kernel (small weight matrix) slides across the input. At every position it computes an element-wise multiply then sums the results into one output number. Click any output cell or press Auto to watch it step through.

Real-World Applications

  • Edge Detection — Sobel/Prewitt kernels are hand-crafted convolutions; CNNs learn better ones
  • Object Detection — YOLO, Faster R-CNN use conv layers to extract feature maps for detecting objects
  • Medical Image Segmentation — U-Net (encoder-decoder CNN) segments tumors, organs, cells
  • Image Enhancement — blur (Gaussian kernel), sharpen, denoise — all are convolution operations

Alternatives & Modern Approaches

  • Strided Convolutions — learnable downsampling, used in modern architectures (replaces pooling)
  • Global Average Pooling — replaces FC layers at the end; used in ResNet, EfficientNet. Reduces overfitting.
  • Adaptive Pooling — produces fixed output size regardless of input dimensions (essential for variable-size inputs)

Modern Additions to the Pipeline

  • Batch Normalization — after every Conv layer; stabilizes training, allows higher learning rates
  • Skip/Residual Connections — ResNet innovation; enables 100+ layer networks by solving vanishing gradients
  • Depthwise Separable Conv — MobileNet; 8-9x fewer parameters, designed for mobile/edge deployment
2.4

Famous CNN Architectures

From 60K to 60M parameters — a timeline of breakthroughs.

LeNet-5 1998

60K params

Established Conv→Pool→Conv→Pool→FC for digit recognition. Deployed in USPS handwriting recognition and ATM check reading.

AlexNet 2012

60M params

Won ImageNet (15.3% vs 26%). Introduced ReLU, dropout, GPU training. Sparked the deep learning revolution — GPU training became standard.

VGG-16 2014

138M params

Proved uniform 3×3 convolutions stacked deep work well. Two 3×3 = one 5×5 receptive field. Still used as feature extractor (VGG-16 backbone) and for neural style transfer.

ResNet 2015

25M params

Skip connections enabled 152+ layers. 3.57% error — surpassed human level (~5.1%). Backbone for object detection (Faster R-CNN), segmentation (U-Net), medical imaging. Most-cited architecture.

Trend: Parameter count does not equal performance. ResNet (25M) beat VGG (138M) with skip connections, proving architecture design matters more than brute-force scaling for CNNs.
