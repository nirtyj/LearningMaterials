4.2

Encoder vs Decoder

Three paradigms — each masks attention differently.

Encoder-Only

Bidirectional — every token sees every other. For understanding/classification.

Examples: BERT, ViT

Real-World Usage

  • Google Search — BERT powers ranking and snippet selection
  • Named Entity Recognition — extracting people, places, orgs from text
  • Sentiment Analysis — classifying review/comment polarity
  • Document Classification — routing, tagging, categorization

Decoder-Only

Causal — tokens only see the past. Generates left-to-right.

Examples: GPT, LLaMA, Claude

Real-World Usage

  • Chatbots — GPT, LLaMA, Claude conversational assistants
  • Code Generation — GitHub Copilot, Cursor
  • Creative Writing — stories, poetry, marketing copy
  • Reasoning — chain-of-thought, math, planning
Dominant paradigm today. Decoder-only scaled to 100B+ parameters is the foundation of modern AI assistants.

Encoder-Decoder

Encoder (bidirectional) + Decoder (causal + cross-attention to encoder).

Examples: T5, Whisper, Original Transformer

Real-World Usage

  • T5 — summarization and translation
  • Whisper — speech-to-text transcription
  • BART — abstractive text generation and denoising
The causal mask: upper-triangular −∞ values added before softmax, forcing future positions to zero attention weight.
4.3

Key Concepts

The building blocks that make transformers trainable and powerful.

Residual Connections

x + f(x) at every sub-layer. Gradient highway through 100+ layers. Without these, GPT-3's 96 layers would be untrainable.

Layer Normalization

Normalize across features (not batch). Keeps activations stable. Pre-LN (before attention) is now standard — more stable than Post-LN.

Position Encoding

Sinusoidal (fixed) or learnable. Without it, transformer is order-blind. RoPE (LLaMA) and ALiBi (BLOOM) enable length generalization beyond training context.

Softmax Temperature

1/√dk prevents large dot products from pushing softmax to extremes. At inference: T<1 for factual tasks, T>1 for creative writing.

Causal Mask

Upper-triangular −∞. Token i only attends to 0..i. Enables autoregressive generation. This is what makes GPT autoregressive — generates one token at a time.

GELU Activation

x·Φ(x) — smooth probabilistic gate in all transformer MLPs. Being replaced by SwiGLU in LLaMA/Mistral for better performance.
