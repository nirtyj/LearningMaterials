1. Building Blocks

The fundamental concepts every architecture builds upon.

1.1

The Neuron

The atom of neural networks — a weighted sum followed by a non-linear activation.

A single neuron takes multiple inputs, multiplies each by a learnable weight, adds a bias, and passes the result through an activation function. This tiny unit, repeated millions of times, is the foundation of every neural network.

Inputs (fixed)
x1 = 0.8    x2 = 0.6
Activation Function
output = relu(0.8*0.50 + 0.6*(-0.30) + 0.10) = relu(0.32) = 0.32

Industry Use: Where the Neuron Concept Appears

  • Logistic Regression — literally one neuron with sigmoid activation; still the baseline classifier in industry
  • Recommendation Systems — user/item embedding dot product + bias is a neuron; Netflix, Spotify use this at scale
  • Binary Classifiers — spam filters, fraud detection, ad click prediction all reduce to neuron-like units at the output
1.2

Activation Functions

Non-linearities that give networks their power.

Without activations, stacking layers just produces another linear transformation. Activation functions introduce non-linearity, enabling networks to learn complex, curved decision boundaries. The choice of activation affects training speed, gradient flow, and model quality.

 
ReLU
f(x) = max(0, x)
Range: [0, ∞)

Default for CNNs (ResNet, VGG). Fast, sparse. Used in hidden layers of most feedforward networks.

Warning: Dying neurons — once a neuron outputs 0 for all inputs, gradients vanish and it never recovers ("stuck at 0").
Sigmoid
σ(x) = 1 / (1 + e-x)
Range: (0, 1)

Output layer for binary classification. Gates in LSTMs/GRUs. Logistic regression.

Warning: Vanishing gradients — for large |x| the gradient approaches 0, making deep networks hard to train.
Tanh
tanh(x) = (ex - e-x) / (ex + e-x)
Range: (-1, 1)

RNNs, LSTMs (cell state). Normalizing outputs to [-1,1]. Some GAN generators.

Warning: Vanishing gradients — same saturation problem as sigmoid, though zero-centered output helps somewhat.
GELU
GELU(x) = x · Φ(x)
Range: ≈(-0.17, ∞)

Default in Transformers: GPT, BERT, ViT, LLaMA. Smooth probabilistic gating.

Warning: More compute than ReLU — involves Gaussian CDF approximation, but modern GPUs handle it efficiently.
Modern Trend: SwiGLU — SwiGLU (Gated Linear Unit with Swish) is replacing GELU in latest LLMs (LLaMA, Mistral, PaLM). It uses x·σ(βx)·(xW₁) gating for better gradient flow.
1.3

Layers & Forward Pass

Composing neurons into layers to form a network.

A layer is a group of neurons that all receive the same inputs. In a fully-connected (dense) layer, every neuron connects to every output of the previous layer. The forward pass pushes data through each layer sequentially: input → hidden → output.

Architecture: [4, 6, 6, 3]
Parameters: 0 weights + 0 biases = 0 total
Dense layers grow quadratically: doubling layer width 4× the parameters. This is why modern architectures prefer sparse structures (convolutions, attention) for most of the network.

Where Dense Layers Are Used

  • Classification Heads — final layers of CNNs, ViTs (e.g., Linear(768 → num_classes))
  • MLP Blocks in Transformers — the FFN after attention: Linear(d→4d) → GELU → Linear(4d→d)
  • Recommendation Systems — user/item embeddings fed through dense layers (YouTube, TikTok ranking)
  • Note: Pure dense networks are rare today — most use dense layers as components within CNNs or Transformers
1.4

Loss Functions

How networks measure their mistakes.

The loss function quantifies the gap between prediction and truth. Training is the process of minimizing this value. Different tasks require different loss functions — the choice shapes what the network learns to optimize.

Scenario: True label y = 1 (positive class). Adjust the predicted probability:
MSE Loss
0.0900
(ŷ − y)²
Cross-Entropy Loss
0.3567
−log(ŷ)
Cross-entropy explodes as ŷ → 0 (confident & wrong), creating a powerful gradient signal that pushes the network to correct confident mistakes. MSE remains gentle, making CE far better for classification.

Which Loss for Which Task

  • MSE → Regression (price prediction, age estimation, weather forecasting)
  • Cross-Entropy → Classification (image recognition, spam detection, sentiment analysis)
  • Binary CE → Multi-label tagging (an image can be both "dog" AND "outdoor")
  • Contrastive Loss → Embedding learning (CLIP, face recognition, sentence similarity)
  • CTC Loss → Sequence alignment (speech recognition, OCR without alignment)
1.5

Backpropagation & Gradient Descent

How networks learn from their mistakes.

Backpropagation computes how much each weight contributed to the error by propagating gradients backwards through the network using the chain rule. Gradient descent then updates each weight in the direction that reduces the loss: w ← w − lr × ∇L.

Step 0 — w = 2.00  |  Loss = (w − 0.5)² = 2.2500
Too small a learning rate: painfully slow convergence. Too large: overshooting the minimum and diverging. Finding the right lr is one of the most important hyperparameter choices in deep learning.

Optimizers in Practice

  • SGD (+ Momentum) — still used for CNNs in research. Good generalization properties.
  • Adam / AdamW — default for Transformers and LLMs. Adaptive learning rates per parameter.
  • Note: AdamW (decoupled weight decay) is the industry standard for training LLMs. Learning rate schedules: warmup → cosine decay.
1.6

Features & Representations

What networks actually learn inside their layers.

Each layer in a deep network transforms raw input into increasingly abstract representations. Early layers detect simple patterns; deeper layers compose them into meaningful concepts. This feature hierarchy is why deep learning works — it automatically discovers the right features for the task.

Layer 1–2
Edges, Colors, Gradients

Simple oriented edges, color blobs, brightness gradients. The most basic visual primitives — similar across all image domains.

Layer 3–4
Textures, Patterns, Corners

Combinations of edges form textures (fur, brick, fabric), repeating patterns, corners, and junctions.

Layer 5–7
Parts: Eyes, Wheels, Fur

Recognizable object parts emerge. Neurons respond to eyes, wheels, ears, windows — semantically meaningful components.

Layer 8+
Objects: Faces, Cars, Dogs

Full object representations. Neurons activate for entire faces, specific dog breeds, car models — task-level concepts.

This hierarchy was famously visualized by Zeiler & Fergus (2013). The key insight: early features are universal (edges work for any image), while later features become domain-specific. This is why transfer learning works.

In Practice: Transfer Learning

  • Pre-trained features (ImageNet) transfer to medical imaging, satellite imagery, manufacturing defect detection
  • Fine-tuning strategy: freeze early layers (edges/textures are universal), retrain later layers for your domain
  • Modern approach: ViT/CLIP pre-training on billions of images → fine-tune on 100 examples works surprisingly well
