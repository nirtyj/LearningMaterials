BFS (deque)

q = deque([start])
seen = {start}
while q:
    node = q.popleft()
    for nei in graph[node]:
        if nei not in seen:
            seen.add(nei)
            q.append(nei)

Sliding window max (monotonic deque)

dq = deque()  # indices, vals decreasing
for i, x in enumerate(nums):
    while dq and nums[dq[-1]] <= x:
        dq.pop()
    dq.append(i)
    if dq[0] <= i - k:
        dq.popleft()
    if i >= k - 1:
        out.append(nums[dq[0]])

Top-k via fixed-size heap

h = []
for x in nums:
    heapq.heappush(h, x)
    if len(h) > k:
        heapq.heappop(h)
# h[0] is k-th largest

Dijkstra (heap of pairs)

pq = [(0, src)]
dist = {src: 0}
while pq:
    d, u = heapq.heappop(pq)
    if d > dist[u]: continue
    for v, w in graph[u]:
        nd = d + w
        if nd < dist.get(v, inf):
            dist[v] = nd
            heapq.heappush(pq, (nd, v))

LRU cache (OrderedDict)

def get(k):
    if k not in cache: return -1
    cache.move_to_end(k)
    return cache[k]
def put(k, v):
    if k in cache: cache.move_to_end(k)
    cache[k] = v

Gotchas

  • defaultdict[k] on a missing key creates it. Use k in dd or dd.get(k) to check without inserting.
  • Counter zero counts: c[x] = 0 does not remove the key. Use del c[x] or +c to strip zero/negative entries.
  • len(c) includes keys with zero or negative counts.
  • c1 - c2 drops zero/negative counts; c1.subtract(c2) keeps them (can go negative).
  • dq.extendleft([1,2,3]) → deque([3,2,1]). Iterable is reversed because each element is appendleft'd.
  • dq[i] middle indexing is O(n). Python docs explicitly say to use a list for fast random access.
  • heappushpop(h, x) pushes then pops smallest. heapreplace(h, x) pops smallest then pushes — different result when x is smaller than the current root.
  • Iterating a heap (for x in h) does not yield sorted order — heap invariant ≠ sorted.
  • set.pop() returns an arbitrary element, not min/max.
  • bisect.insort finds the position in O(log n) but the list-shift is O(n). For frequent sorted inserts, use SortedList.
  • itertools.groupby only groups consecutive equal items. Sort first if you want full grouping.
  • str.find(sub) returns -1 if not found; str.index(sub) raises ValueError. Same pattern for list.index.
  • Convert defaultdict to plain dict before returning if a grader checks structure: dict(graph).
  • list.pop(0) is O(n) — for queue-like access from the front, use deque.popleft().
  • Tuples sort lexicographically. Use (priority, tiebreaker, payload) in heap entries to avoid comparing un-orderable payloads.
