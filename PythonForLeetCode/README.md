# Python for LeetCode

Visual guide and practice notebook for LeetCode-style coding interviews in Python.

## How to use

Open **[`index.html`](./index.html)** in a browser. Each topic is its own page so you can read one data structure end-to-end (initialization, operations, common interview patterns).

### Topics

- **Foundations** — Compilation & Execution, Variables & Memory, Time Complexity
- **Data Structures** — List, Dict, Set, Stack, Queue, Heap, String, DefaultDict / Counter
- **OOP** — Node defs, dataclass, dunder, sorting, abstract, OOD building blocks
- **Patterns & Reference** — Algorithm Patterns, Python Power Features, Python Builtins, Common Gotchas, Methods by Operation

## Practice

- **Hands-on drills**: [`practise/list.py`](./practise/list.py), [`practise/dict.py`](./practise/dict.py) — fill-in-the-blank Python exercises that mirror every template in the guide.
- **Notebook**: [`python_leetcode_practice.ipynb`](./python_leetcode_practice.ipynb) or open in [Colab](https://colab.research.google.com/github/nirtyj/LearningMaterials/blob/main/PythonForLeetCode/python_leetcode_practice.ipynb).

## Quick reference

- [`python_cheatsheet.md`](./python_cheatsheet.md) — pocket-sized list of patterns that come up most often.
