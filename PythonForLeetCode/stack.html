Operations

stack = []

# ── Push / Pop / Peek ──
stack.append(1)              # push → [1]
stack.append(2)              # push → [1,2]
stack.append(3)              # push → [1,2,3]
stack[-1]                    # peek → 3 (no remove)
stack.pop()                  # pop → 3, stack=[1,2]

# ── Check ──
len(stack) == 0              # is empty?
not stack                    # Pythonic empty check

# ── Common Pattern: Monotonic Stack ──
stack = []
for i, num in enumerate(nums):
    while stack and nums[stack[-1]] < num:
        stack.pop()
    stack.append(i)

🛠️ Built-ins commonly paired with stacks

When you write stack code in interviews, these are the built-ins you reach for most.

# ── Building ──
stack = []                                # plain list
stack = list(arr)                          # initialize with elements
stack = [root] if root else []             # conditional init (DFS start)

# ── Push / pop / peek (built-in idioms — no method calls needed) ──
stack.append(x)                           # push
stack.pop()                               # pop top
stack[-1]                                 # peek — no method needed
not stack                                 # Pythonic empty check
len(stack) == 0                           # explicit length check

# ── Iterating without consuming ──
for x in stack: ...                       # bottom-to-top order
for x in reversed(stack): ...             # top-to-bottom — what you usually want
while stack and stack[-1] < x:            # monotonic-stack scan idiom
    stack.pop()

# ── Snapshot / convert ──
list(stack)                               # shallow copy
"".join(stack)                            # build string from char stack — common in parsing

Interview Pattern: Monotonic Stack

LIFO for matching, next greater / smaller. Used in: Valid Parentheses, Daily Temperatures, Largest Rectangle in Histogram, Car Fleet

🐍 Python Shortcuts
# ── Valid Parentheses (LC 20) ──
def isValid(s):
    stack = []
    pairs = {')': '(', ']': '[', '}': '{'}  # ← dict for O(1) match
    for c in s:
        if c in pairs:
            if not stack or stack[-1] != pairs[c]:
                return False
            stack.pop()
        else:
            stack.append(c)
    return not stack                   # ← Pythonic empty check

# ── Daily Temperatures (LC 739) — Monotonic stack ──
def dailyTemperatures(temps):
    ans = [0] * len(temps)
    stack = []                           # stores indices
    for i, t in enumerate(temps):
        while stack and temps[stack[-1]] < t:
            j = stack.pop()
            ans[j] = i - j               # ← days to wait
        stack.append(i)
    return ans

# ── Python tricks used ──
# • stack[-1]          — peek (no method needed!)
# • not stack          — Pythonic "is empty" check
# • dict for matching  — O(1) bracket pair lookup
